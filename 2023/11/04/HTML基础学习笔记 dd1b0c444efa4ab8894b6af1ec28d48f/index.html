HTML基础学习笔记

一.HTML语法格式

1.标记规则

在标记的过程中，如果使用了某个开始标签，通常需要使用对应的标签进行结尾。

但并非所有的标签都是如此，HTML中还存在一些元素，它们仅由开始标签构成。

2.单标签组成的元素

这类标签不需要内容产生效果，通常表示对网页的某种行为，它们不用标记任何内容，开始即是结束。

3.属性

属性可以为网页元素提供更多信息， 它总是以属性名/值对的形式出现。

HTML 属性大致分为下面两种类型：

通用属性：适用于大部分或者所有 HTML 标签；

专用属性：只适用于一个或者几个特定的 HTML 标签。

4.构成网页的元素

doc-document文件

type-类型

5.网页头和网页体

网页头（又称为网页头部信息）向浏览器提供文档的一些信息。

缺少<html>、<head>、<body>等标签，网页不会报错，但是会让网页的加载速度变慢。

二.网页中的文本内容

1.标题与段落

heading-标题

  • h1与title的区别

    title 元素指的是网页的标题名称。它被显示在浏览器的标签页中，且一个HTML文档只能有一个title标签。

    h1 元素指的是网页内容中的标题，一个网页可以含有多个h1标题。

2.段落元素

  • p-paragraph-段落
  • 段落与段落之间有默认的间隙
  • 一个段落中的文字会根据浏览器窗口的大小自动换行
  • 为元素添加内容时，若出现换行或多个空格，浏览器会将其转换为一个空格

  • 强制转换

    1.br-break-打断

    2.若想要强制在某个位置换行，可以使用<br>标签。

3.利用属性定位元素

  • 属性分类
  1. class 属性用来对相同的元素进行分类。
  2. 它的属性值就是每个分类的名称（类名）。名称可以设定为任意内容且该名称可以重复使用
  3. class 可以设置在任何元素中，是一个通用属性
  • 设定唯一标识
  1. id属性为某个元素分配一个唯一的标识符
  2. id属性的值是唯一的
  • 文本格式化
  1. 加粗，倾斜，删除线，下划线
  2. 将想要格式化的文字作为内容，放在相应的标签中即可
  • 显示结构分为两类

第一类是类似于p元素的块级元素

  • p元素

    1.块级元素最主要的特点是它自己独占一行

    2.块级元素在浏览器显示时，通常会以新的一行开始

    3.块级元素中可以包含其他的行内或块级元素

第二类是类似于b元素的行内元素

  • b元素

    1.会与其他元素在同一行显示

    2.不能包含块级元素

三.超链接与图片

1.图片元素

网页加载图片的过程可以分为三个部分：

第一步，浏览器从html中获取图片的地址，即知道图片在哪台电脑（服务器）、哪个文件夹中

第二步，浏览器根据地址查找图片文件

第三步，根据查找结果，决定图片加载成功或失败

img标签

  • img 元素可以为网页插入一张图片，它是一个由单标签组成的行内元素
  • img-image-图像
  • alt-altemate-代替的
  • src-source - 来源

设置图片宽高

  • 可以只设置图片的宽度（或高度），浏览器会按照比例自动调整图片的高度（或宽度）

获取图片的地址

1.当图片与HTML文档在同一个文件夹时

图片地址的格式为

./文件名称.文件类型

  • .代表文档所在的文件夹，/用来分割文件夹与文件的名称

可简写为文件名称.文件类型

2..当图片与HTML文档在不同的文件夹时

文件夹中的图片地址格式

文件夹名称/文件名称.文件类型

3.网络中的图片

对图片点击右键，选择【复制图片地址】

2.超链接

超链接（或简称链接），是指从一个网页指向另一个目标的连接关系

一个完整的网站应该由多个不同的html文档相互链接而成

  • <a>标签表示一个超链接，它是一个双标签，内容以</a>结束
  • 它是一个行内元素
  • a-anchor-键
  • href-Hypertext Reference-引用超文本
  • href属性中的地址可以包含:
  1. 另一个网页
  2. 网页中的某个元素
  3. 一张图片，一个JS程序等等

描述内容
一段被超链接标记的文字。

用户通过点击这段文字进入超链接

当超链接被点击时，浏览器默认直接从当前页面进入目标页面。

利用target属性可以修改这项规则

该属性常用的值有两种：

  • _blank，浏览器会在新的页面中打开文档；
  • _self，浏览器会在当前页面打开文档（默认）。

内部超链接

  • 使用超链接不仅能够指向一个html文档，还可以链接一个含有id属性的元素
  • 只要将id值作为href的地址即可，格式为href="#元素的id"

结合图片的超链接

  • 将img元素作为内容，放在a元素中。即可为图片添加一个超链接
